# colorful-scroll

A 2D side-scrolling platformer game inspired by Mario. It will feature multiple levels, enemies, power-ups, and a scoring system. The game's visual style will be vibrant and colorful, with easy-to-navigate controls designed for a fun and engaging user experience.

## Setup

```sh

```


This will run a dev server with auto reloading and an instant preview.
